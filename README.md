# Workflow Engine

Execution engine for multi-agent workflows.

## Features

- **Workflow Execution** - Run complex workflows
- **State Management** - Track workflow states
- **Parallel Execution** - Execute tasks in parallel
- **Error Handling** - Handle workflow errors

## Quick Start

```python
from workflow_engine import WorkflowEngine

engine = WorkflowEngine()
engine.load("workflow.json")
result = engine.run()
```
